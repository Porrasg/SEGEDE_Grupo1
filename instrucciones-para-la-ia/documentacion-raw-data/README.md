# Documentación y Datos en Crudo (Raw Data)

Esta carpeta (`documentacion-raw-data/`) está destinada para que el desarrollador guarde **manualmente** archivos de referencia, requerimientos de negocio, diagramas de bases de datos, manuales, notas o especificaciones funcionales del sistema.

## 📖 Instrucciones para la IA
- Cualquier archivo colocado dentro de esta carpeta debe ser considerado por la IA como **material de apoyo de alta prioridad** para entender mejor las reglas de negocio, el alcance y los objetivos técnicos de **SEGEDE_Grupo1**.
- Cuando el usuario solicite implementar una nueva funcionalidad o definir una nueva entidad, la IA deberá verificar primero si existen documentos de referencia en este directorio antes de tomar decisiones de diseño o consultar dudas.
